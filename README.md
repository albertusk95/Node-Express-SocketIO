# NodeJS-ExpressJS-SocketIO

Beforehand, please install these modules by yourself.

### Required modules
> * express
> * body-parser
> * pug 
> * socket.io
> * underscore

### Installation
Open your terminal:
> npm install express<br />
> npm install body-parser<br />
> npm install pug<br />
> npm install socket.io<br />
> npm install underscore<br />
